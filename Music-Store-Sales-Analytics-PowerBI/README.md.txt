# 🎵 Music Store Sales Analytics Dashboard | Power BI Project


## 📌 Project Overview

This project focuses on analyzing music store sales data to generate meaningful business insights using **Power BI and MySQL**.

The main objective of this project is to understand sales performance, customer behavior, and music trends through an interactive dashboard.

The data was imported from a MySQL database and transformed into an analytical dashboard using Power BI.


---

# 🛠️ Tools & Technologies Used

- MySQL 8.0.37
- Power BI Desktop
- SQL
- DAX (Data Analysis Expressions)
- Data Modeling

---

# 📊 Dashboard Overview

The Power BI dashboard provides insights into:

### 💰 Sales Analysis
- Total Sales
- Revenue performance
- Sales trends


### 👥 Customer Analysis
- Customer distribution
- Country-wise sales analysis
- Customer purchasing behavior


### 🎶 Music Performance Analysis
- Top performing artists
- Popular genres
- Best-selling tracks


---

# 📈 Key KPIs Created

The dashboard includes:

- Total Sales
- Total Orders
- Total Customers
- Tracks Sold
- Average Order Value

# 🔍 Data Analysis Performed

### Data Preparation

- Connected MySQL database with Power BI
- Loaded multiple tables
- Created relationships between tables
- Prepared data model for analysis


### Data Modeling

Relationships created between:

- Customers
- Invoice
- Invoice Line
- Tracks
- Albums
- Artists
- Genres
- Media Type


### DAX Measures

Created calculated measures for:

- Revenue calculation
- Order count
- Customer count
- Track sales
- Average order value
